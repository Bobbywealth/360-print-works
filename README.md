# 360 Print Works

A comprehensive printing company website with landing page and CRM backend.

## Tech Stack
- Next.js 16 with App Router
- TypeScript 5
- Tailwind CSS 4
- shadcn/ui components
- Prisma ORM with SQLite
- NextAuth.js

## Quick Start

```bash
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the landing page.
Visit [http://localhost:3000/crm](http://localhost:3000/crm) for the CRM dashboard.
